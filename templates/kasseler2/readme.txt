============================================
index.tpl - Основной шаблон оформления
============================================
$title         - Заголовок страницы
$meta          - Мета информация страницы
$link          - Загружаемые стили
$script        - Загружаемые jsvascript-ы
$topbaner      - Верхний баннер
$footbaner     - Нижний баннер
$block_left    - Левые блоки
$block_center  - Верхние блоки
$block_down    - Нижние блоки
$block_right   - Правые блоки
$message       - Сообщение на главной
$modules       - Загружаемый модуль
$time          - Информация о скорости генерации страницы
$license       - Вывод копирейтов системы  (является обизательной переменной)
$var_info      - Информация о супер глобальных переменных
$query_info    - Информация о запросах в БД
$footer        - Дополнительные скрипты описанные в includes/header.php
$load_tpl      - Загружаемый шаблон
$user_name     - Имя пользователя

Дополнительные варианты шаблона: index[-название_модуля[-идентификатор]].tpl
index-home.tpl - Шаблон для главной страницы сайта

HTML коментарии: 
<!--BEGIN leftblock-->, <!--END leftblock--> - Отделяет левые блоки от остального HTML в шаблоне (используется для отключения левых блоков)
<!--BEGIN rightblock-->, <!--END rightblock--> - Отделяет правые блоки от остального HTML в шаблоне (используется для отключения правых блоков)

Дополнительные возможности шаблонизатора
<guest></guest>  - Содержания внутри тегов видят только гости
<user></user>    - Содержания внутри тегов видят только пользователи
<moder></moder>  - Содержания внутри тегов видят только модераторы
<admin></admin>  - Содержания внутри тегов видят только администраторы

============================================
PHP вставки
============================================
В любой шаблон системы есть возможность php вставок
Припер: 
<?php
global $userinfo
echo $userinfo['user_name'];
?> 
Данная вставка выведет имя пользователя
